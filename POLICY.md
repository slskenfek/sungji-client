# Dungeon Survivors Policy

## Core Concept

- 플레이어는 `궁수`, `전사`, `마법사`, `거너` 중 하나를 선택한다.
- 플레이어는 던전의 방 단위 스테이지에 입장한다.
- 전투는 `자동 공격`과 `키보드 이동 조작` 중심으로 설계한다.
- 몬스터는 웨이브 또는 지속 스폰 방식으로 끝없이 몰려온다.
- 몬스터 처치로 경험치를 획득하고, 경험치 누적으로 레벨업한다.
- 레벨업은 생존력, 화력, 범위, 공격 속도 같은 성장 선택으로 이어진다.
- 각 스테이지 웨이브 종료 시 일반 보스가 등장하며, 상단 버튼으로 별도의 `최종 보스`를 즉시 소환할 수 있다.
- 보스를 처치하면 현재 방을 클리어한다.
- 방 클리어 시 장비와 보상을 획득한다.

## Prototype Scope

- 현재 프로토타입은 `이동`, `카메라 추적`, `기본 캐릭터 애니메이션`, `체력바`, `경험치바`, `미니맵`, `룸 UI` 표시를 기본 범위로 둔다.
- 장기 컨셉상 직업군은 네 가지지만, 현재 프로토타입 구현은 `전사` 단일 캐릭터에 집중한다.
- 룸 구조는 추후 방 클리어, 보스 스폰, 보상 선택 UI와 연결한다.
- 현재 구현은 `전사`로 시작해 26레벨에 `검투사`로 전직하는 성장 구조다.
- 전사는 `근접 1칸 공격`과 `자동 기본공격`을 기준으로 설계한다.
- 전사의 기본 능력치는 `데미지 10`, `힘 5`, `덱스 2`, `지식 1`, `방어 7`이다.
- 전사는 레벨업 1회마다 `힘 +1`, `방어 기본치 +0.3`, `데미지 기본치 +0.7`이 적용된다.
- 전사는 레벨이 `4`의 배수일 때 `덱스 +1`, `5`의 배수일 때 `지식 +1`이 추가로 적용된다.
- `힘`이 1 증가하면 `데미지 +0.65`, `최대 체력 +5`가 증가한다.
- `덱스`가 1 증가하면 `공격속도 +0.12`, `방어 +0.3`이 증가한다.
- `지식` 1당 `마법 데미지 7`을 제공한다.
- 25레벨 이후 전직 시 기존 능력치·장착 장비·강화 수치·인벤토리를 유지한 채 레벨과 경험치를 `LV 1` 기준으로 초기화하고, 순수 힘은 `6배`, 덱스는 `10배`, 지식은 `5배`, 계산된 데미지는 `2배`가 된다.
- 검투사는 쌍검을 사용하며 기본 공격 대상 주변으로 번개가 최대 5회 전이되어 각 대상에 기본 데미지의 `30%`를 입힌다.
- 검투사는 전사보다 기본 이동속도가 `3배`, 기본 공격 사거리가 `2배` 증가한다.
- 전직 후 검투사가 다시 `LV 10`에 도달하면 기본 공격에 화면 흔들림, 짧은 히트 스톱, 충격파와 불꽃 타격 효과가 추가된다.
- 검투사의 기본 공격은 `10%` 확률로 주 대상과 연쇄 번개 데미지가 `200%`가 되는 크리티컬을 발생시키며, 더 강한 충격 효과와 `크리티컬!` 텍스트를 표시한다.
- 검투사가 `LV 25`에 도달하면 능력치·장비·인벤토리를 유지하고 힘 `100배`, 덱스 `50배`, 지식 `10배`를 적용한 뒤 `LV 1`의 `검투사 2차`로 다시 환생한다.
- 검투사 2차는 `CTRL` 악마모드 사용 직후 `2초` 동안 기본 공격 크리티컬 확률이 `100%`가 된다.
- 검투사의 `CTRL` 악마모드는 재사용 대기시간 `10초`, 지속시간 `5초`이며 공격속도 `300% 증가`, 현재 이동속도에서 `20% 증가`, 입힌 피해의 `20%` 생명력 흡수를 제공한다.
- 높은 레벨 몬스터를 처치하면 경험치는 몬스터 레벨마다 `1.2배`씩 더 많이 획득한다.
- 몬스터는 현재 프로토타입에서 `원거리 공격`을 사용하되, 플레이어를 직접 추격하면서 사거리 안에서도 압박을 유지한다.
- 몬스터는 레벨이 오를수록 방어 효율이 상승해 더 오래 버틴다.
- 무기, 투구, 갑옷, 신발은 같은 장비로 강화 시도 시 `25%` 확률로 재료 장비 능력치의 `30%`를 흡수해 강화된다.
- 장신구는 이름과 무관하게 `accessory` 타입이 같으면 같은 방식으로 계속 강화된다.
- 강화 재료는 성공 여부와 관계없이 소모되며 성공·실패 결과를 화면에 표시한다.
- 아이템 등급은 낮은 순서대로 `레어`, `에픽`, `전설`, `신화`, `최강`의 5단계만 사용한다.
- 일반 보스는 `신화`, 최종 보스는 `최강` 등급 장비를 확정 드롭한다.
- 장착한 장신구가 강화 `+5`에 도달하면 기본 공격이 `120px` 화염 스플래시 공격으로 바뀌고 기본 공격 피해가 `20%` 증가한다.
- 보스가 드롭한 `신화` 장비를 강화 재료로 사용하면 재료 능력치의 `120%`를 흡수한다.
- 화면 상단의 `최종 보스 소환` 버튼은 현재 웨이브를 종료하고 현재 스테이지 일반 보스 대비 체력 `10배`, 투사체 데미지 `2배`, 크기 `1.45배`의 최종 보스를 소환한다.
- 최종 보스가 드롭한 `최강` 장비는 동일 장비 강화 시 성공률 `100%`, 재료 능력치 흡수율 `150%`를 적용한다.
- 최강 무기 강화 효과는 검을 붉은 화염검으로 변경하고 최종 데미지를 `3배`로 만든다.
- 최강 신발 강화 효과는 날개 외형을 추가하고 최종 이동속도를 `2배`로 만든다.
- 최강 장신구 강화 효과는 검투사 연쇄 번개 스플래시 데미지를 기본 공격 데미지의 `120%`로 만든다.
- 최강 갑옷 또는 투구 강화 효과는 방어 계산 후 받는 물리 투사체 피해를 `10%` 감소시킨다.
- 보스 처치 후 플레이어가 `다음 스테이지 넘어가기`를 선택하면 다음 스테이지가 시작된다.
- 다음 스테이지에서도 플레이어 레벨, 경험치, 장착 장비, 장비 강화 수치, 인벤토리를 모두 유지한다.
- 스테이지가 오를수록 몬스터 레벨·체력·방어·공격력·이동 속도와 경험치·아이템 성능이 함께 상승한다.
- 몬스터 한 마리의 경험치는 현재 필요 경험치의 `35%`, 보스 경험치는 `100%`를 넘지 않는다.
- `6`의 배수 스테이지 보스는 일반 스테이지 보스 대비 체력 `5배`, 투사체 데미지 `1.2배`가 추가 적용된다.

## UX Rules

- 메인 플레이는 한 손 또는 양손 키보드 조작만으로 성립해야 한다.
- 전투 판단은 수동 에임보다 `이동`, `회피`, `포지셔닝`에 무게를 둔다.
- UI는 생존 상태, 성장 상태, 현재 방 진행도를 항상 즉시 이해할 수 있게 유지한다.

## Maintenance Rules

- 게임 컨셉, 전투 규칙, 성장식, 직업 우선순위가 바뀌면 이 문서를 같은 작업에서 즉시 갱신한다.
- 구현 코드가 바뀌어 정책과 실제 동작이 어긋나면 정책 문서를 최신 구현 기준으로 자동 정리한다.
- 시스템 파일은 `config`, `scene`, `entity`, `ui`, `system` 책임 단위로 분리해 유지한다.
